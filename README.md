# F-18 Simulation

F-18 simulation in low angle of attack using Simulink

Aerodynamic data taken from the book "Advanced Flight Dynamics with Elements of Flight Control" by Nandhan K Sinha and N Ananthkrishnan

This simulation covers the low angle of attack dynamics of F18

|Wing span, $b$|11.405|$m$|
|Mean aerodynamic chord, $c$|3.511|$m$|
|Wing planform area, $S$|37.16|$m^2$|
